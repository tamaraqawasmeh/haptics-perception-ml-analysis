# Haptics Perception Analysis

This repository contains the analysis notebook for a study examining user perception and performance in response to haptic (tactile) feedback.

The project evaluates how haptic stimuli influence user experience, perception, and task outcomes, using quantitative analysis and visualization methods implemented in Python.

---

## Overview

Haptic technology plays a critical role in modern human–computer interaction, enabling communication through touch in applications ranging from virtual reality and robotics to accessibility tools and medical interfaces.

This analysis investigates experimental results related to haptic feedback, focusing on measurable outcomes such as performance metrics, response patterns, and perceptual differences across conditions.

---

## Objectives

The primary objectives of this analysis are:

- To process and clean experimental data from a haptics study  
- To evaluate differences in user responses across experimental conditions  
- To compute descriptive and inferential statistics  
- To generate visualizations that support interpretation of results  

---

## Repository Contents

- `haptics_perception_analysis.ipynb` — Jupyter notebook containing the full analysis workflow

---

## Methodology

The notebook performs:

- Data loading and preprocessing  
- Statistical analysis  
- Visualization of experimental outcomes  
- Interpretation of key performance indicators  

All analyses are implemented using standard scientific Python tools.
